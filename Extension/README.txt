SOTI AI Analyser Extension
==========================

A powerful browser extension for AI-assisted log analysis and case management.

HOW TO INSTALL
--------------

For Google Chrome & Microsoft Edge:
1. Download or clone this repository to your computer.
2. Open your browser and navigate to the Extensions page:
   - Chrome: Type "chrome://extensions" in the address bar.
   - Edge: Type "edge://extensions" in the address bar.
3. Enable Developer Mode using the toggle switch.
4. Click the "Load unpacked" button.
5. Select the "Extension" folder from this repository.
6. The SOTI AI Analyser should now appear in your extensions list!

For Other Browsers:
Most modern browsers (Brave, Opera, Vivaldi) follow the same steps as Chrome. Look for the "Extensions" settings and "Developer Mode" to load the unpacked folder.

---

STANDALONE OPTION (No Installation Required)
--------------------------------------------

If you prefer not to install the extension, you can run the tool as a standalone web page:

1. Navigate to the "Extension" folder.
2. Locate "SOTI_AI_Analyser.html".
3. Right-click it and select "Open with" (choose your browser), or simply drag and drop the file into an open browser tab.

This will open the SOTI AI Analyser interface in a standard browser tab.

FEATURES
--------
- Log Forensics: Deep analysis of attached log files.
- Case Management: Automated case data synchronization.
- JIRA Integration: Easily create JIRA reports from diagnostic results.
- Privacy: Rigorous PII scrubbing for international data security.
